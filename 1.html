HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a seb server of from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originallt included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. with HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
